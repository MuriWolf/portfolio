Habilidades

Abaixo estão as tecnologias que tenho experiência e conhecimento, algumas mais que outras, porém, já fiz pelo menos um projeto usando cada uma delas.

Passe o mouse sobre o ícone para ver a descrição da tecnologia.

HTML

(Hypertext Markup Language) Linguagem de marcação usada como base de sites.

Conheço as principais tags, funcionamento de formulários, tags semânticas e acessibilidade.

Bootstrap

Framework CSS amplamente usado.

Conheço suas principais utilidades e como usá-las (layout, componentes, utilidades, conteúdo e etc..).

CSS

(Cascading Style Sheet) Linguagem de marcação usada na estilização de sites.

Conheço as principais propriedades, sistema grid, flexbox, animações, design responsivo e estrutura CSS.

Github

Serviço que hospeda sistemas que utilizam o Git.

Entendo sua utilidade e uso.

Javascript

Linguagem de programação usada em sites para adicionar mais interação e dinamicidade.

Conheço os principais usos, funções e utilidades. Sei as atualizações do Ecmascript 6 (como promessas, async, modules, arrow functions e etc...).

Gulp

Pacote Javascript que serve de automação de alguns processos do Front-end.

Utilizando ele consigo: transformar SASS/SCSS em CSS, minificá-lo e mapeá-lo, minificá-o Javacript e mapeá-lo, entre outras pequenas coisas.

NPM

(Node package manager) Pacote de gerencimento de módulos Javascript.

Conheço sua utilidade e funcionamento.

PHP

(Hypertext Preprocessor) Linguagem de programação que roda no lado do servidor, serve para deixar o site mais dinâmico.

Conheço sua sintaxe, principais métodos e propriedades, sei fazer interação com formulários e outras coisas básicas.

SASS

Pré-processador CSS que adiciona algumas funcionalidades.

Uso o pré-processador SASS com a sintaxe SCSS, conheço as principais utilidades e estruturação.

Git

Sistema de controle de versões, usado principalmente no desenvolvimento de software.

Sei seus principais comandos e funcionalidades como: enviar as atualizações para o github, clonar repositórios e etc.

NodeJS

Basicamente o Javacript rodando no lado do servidor ao invés do lado do cliente.

Conheço o básico.

Python

Linguagem de programação utilizado em Big Data, Inteligencia Artificial, Automação, Web e etc.

Já Estudei ela, porém, não mexo mais. Sei sua sintaxe, principais usos, já estudei um pouco de análise de dados, automação e Django, porém, não me aprofundei.
